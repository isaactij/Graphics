This project is a game in which you dodge some enemies and shoot others in order to reach a final destination, which is the victory condition. The project uses a mario theme. In order to move, use the arrow keys, and press up to jump. In order to shoot fireballs, press 'f'. Not all enemies can be shot with fireballs to be killed. In order to win the game, make it to the flag in the end.

Press 'p' to pause
press 'v' to toggle background music
